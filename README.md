
# UART FPGA Implementation (DE0-Nano)

This repository contains the Verilog RTL implementation of a UART transceiver designed for FPGA.

Course: EN2111 Electronic Circuit Design

## Modules

- UART Transmitter
- UART Receiver
- UART Core
- Top Module
- Hex to 7-Segment Decoder

## Simulation

The design was verified using a Verilog testbench and simulated to confirm correct UART frame transmission and reception.

## Hardware Implementation

The design was implemented on a DE0-Nano FPGA board and verified using 7-segment display output.
